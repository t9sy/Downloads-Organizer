# DownloadsOrganizer

**DownloadsOrganizer** is the perfect cure for a digital mess. If your `Downloads` folder is overflowing with thousands of random files, images, and installers, this script will clean it up instantly.

## Features
- Scans your Downloads folder and groups files by their type/extension.
- Automatically creates clean sub-folders: `Images`, `Documents`, `Installers`, `Archives`, `Audio`, and `Video`.
- Moves matching files into their respective folders instantly, keeping your main Downloads area clutter-free.

## Usage
1. Right-click `OrganizeDownloads.ps1`.
2. Select **Run with PowerShell**.
3. Watch as it instantly sorts all your loose files into clean category folders!
